# Support

## Getting help

- For contract bugs, use the bug report issue template.
- For scoped implementation work, use the maintainer task template.
- For docs and setup improvements, use the documentation template.

## Before opening an issue

- Read [CONTRIBUTING.md](./CONTRIBUTING.md).
- Review [docs/open-source/issue-backlog.md](./docs/open-source/issue-backlog.md).
- Confirm the work belongs in the contracts repo and not the client or server repo.

## Project boundaries

This repository covers on-chain creator registration, key supply, pricing, and fee logic on Stellar Soroban. Frontend UX belongs in `accesslayer-client`. Metadata, indexing, and gated off-chain access belong in `accesslayer-server`.
